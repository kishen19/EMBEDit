# EMBEDit
A library for generating high-quality data embeddings for various downstream tasks

## Installation

### 1. Install PyTorch (Hardware Specific)
Install the version of PyTorch apt for your hardware from the [Official PyTorch Website](https://pytorch.org/get-started/locally/). 

### 2. Install Other Dependencies
Run the following command to install the rest of the dependencies.
```bash
python3 -m pip install -r requirements.txt
